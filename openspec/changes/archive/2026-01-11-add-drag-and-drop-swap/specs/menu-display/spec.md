## ADDED Requirements
### Requirement: Drag and Drop Swapping
The system SHALL allow users to swap meals between days using drag and drop.

#### Scenario: Swap Monday and Wednesday
- **WHEN** the user drags the "Lundi" tile and drops it onto the "Mercredi" tile
- **THEN** the meal originally on Monday moves to Wednesday.
- **AND** the meal originally on Wednesday moves to Monday.
- **AND** the background images are updated accordingly.
