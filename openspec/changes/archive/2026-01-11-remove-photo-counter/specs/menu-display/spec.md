## MODIFIED Requirements
### Requirement: Sidebar Display
The sidebar SHALL display the photo slideshow and current date/time information, strictly minimal.

#### Scenario: Clean sidebar
- **WHEN** the sidebar is rendered
- **THEN** it does NOT display photo captions.
- **AND** it does NOT display status text ("Diaporama actif").
- **AND** it does NOT display image counters (e.g. "1 / 5").
