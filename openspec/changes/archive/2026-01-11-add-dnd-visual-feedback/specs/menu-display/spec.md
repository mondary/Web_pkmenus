## MODIFIED Requirements
### Requirement: Drag and Drop Swapping
The system SHALL allow users to swap meals between days using drag and drop, providing clear visual feedback during the process.

#### Scenario: Visual feedback during drag
- **WHEN** the user starts dragging a tile
- **THEN** that tile becomes partially transparent.
- **AND** when hovering over another tile, the target tile shows a visual highlight (e.g., border or background change).
